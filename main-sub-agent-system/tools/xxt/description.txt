超星学习通自动答题工具。通过 playwright 操作浏览器，支持登录、爬取题目、填充答案、提交作业、截图、检查填充状态。

子命令：
- login:     打开浏览器等待用户登录（首次使用或会话过期时调用）
- crawl:     爬取作业页面，提取所有题目（选择题/填空题/判断题）
- fill:      根据答案填充作业（直接传 answers 参数，无需保存文件）
- submit:    暂存并提交作业
- screenshot: 对当前页面截图
- check:     检查各题目的填充状态

推荐工作流（无需保存文件）：
1. crawl → 获取题目（返回 JSON 格式题目列表）
2. Agent 根据题目生成答案
3. fill → 直接传 answers 参数填充答案（如 answers='{"1":"C","2":"A"}'）
4. check → 确认填充结果
5. submit → 提交

答案格式：{"题号": "答案"}，如 {"1":"C","2":"A","3":"答案文本"}

注意：
- fill 命令的 answers 参数支持直接传 JSON 字符串，无需先保存到文件
- answers 通过环境变量传递，无命令行转义问题
- 如果答案量很大，也可以用 answers_file 参数指向文件
