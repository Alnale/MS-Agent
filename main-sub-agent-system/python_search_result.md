# Python 是什么？

**Python** 是一种广泛使用的**高级编程语言**，由荷兰程序员 Guido van Rossum 于 1991 年创建。

### 核心特点

| 特点 | 说明 |
|------|------|
| **简单易学** | 语法接近自然英语，非常适合编程初学者 |
| **解释型语言** | 无需编译，写完代码直接运行 |
| **跨平台** | 支持 Windows、macOS、Linux 等系统 |
| **开源免费** | 社区活跃，拥有海量第三方库 |

### 主要应用领域

- 🤖 **人工智能 / 机器学习**（TensorFlow、PyTorch）
- 📊 **数据分析**（Pandas、NumPy）
- 🌐 **Web 开发**（Django、Flask）
- 🔧 **自动化脚本 / 运维**
- 🕷️ **网络爬虫**
- 🎮 **游戏开发**、科学计算等

### 一句话总结

> Python 因其**简洁优雅的语法**和**强大的生态**，成为目前全球最流行的编程语言之一，尤其在 AI 和数据科学领域占据主导地位。
